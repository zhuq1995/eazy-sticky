# Vue 3 + Electron 便签项目完整提示词

## 🎯 **项目总体描述**

```
作为资深Vue 3和Electron开发专家，请创建一个现代化的桌面便签应用。

技术栈要求：
- 前端框架：Vue 3 Composition API + <script setup>语法
- 构建工具：Vite + electron-vite插件
- 状态管理：Pinia
- 工具库：VueUse for Electron集成
- 样式：原生CSS + CSS变量
- 语言：TypeScript

核心功能：
1. 多便签管理：创建、删除、拖拽移动
2. 窗口置顶：始终显示在最前面
3. 真实便签UI：仿纸质视觉效果
4. 数据持久化：自动保存到本地
5. 系统集成：托盘、快捷键、多显示器

输出要求：完整的项目结构、TypeScript配置、所有源码文件
```
